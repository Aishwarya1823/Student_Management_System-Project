#Student Management System (Spring Boot + Thymeleaf + MySQL)

A real-world Spring Boot full-stack project designed to manage students, course enrollments, and academic records efficiently. The application follows a clean layered architecture and includes form validation, pagination, authentication, and responsive user interfaces.

This repository is public for learning purposes.

---

## 🚀 Tech Stack

* Java (JDK 17/21)
* Spring Boot
* Spring MVC
* Thymeleaf
* Spring Data JPA (Hibernate)
* Spring Security
* MySQL
* Maven / Gradle

---

## ✨ Features

* ✅ Student CRUD (Create, Read, Update, Delete)
* ✅ Course Enrollment Management
* ✅ Server-Side Form Validation
* ✅ Pagination and Sorting
* ✅ Login & Authorization using Spring Security
* ✅ Global Exception Handling
* ✅ DTO and Entity Separation (Clean Architecture)
* ✅ Responsive UI with Thymeleaf Templates
* ✅ Dashboard for Quick Overview
* ✅ Layered Architecture (Controller, Service, Repository)

---

## 📌 Project Objectives

* Simplify student and course management processes.
* Provide secure access with authentication and authorization.
* Demonstrate Spring Boot best practices and clean architecture.
* Offer a real-world learning project for beginners and students.

---

## 🏗️ Architecture

The project follows a layered architecture:

* Controller Layer
* Service Layer
* Repository Layer
* DTO Layer
* Entity Layer
* Database Layer (MySQL)

---

By exploring this project, you will learn:
